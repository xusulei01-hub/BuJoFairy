# Changelog

All notable changes to 旅行手帐 will be documented in this file.

## [0.1.0] - Unreleased

### Added
- 项目初始化：iOS (SwiftUI + SwiftData) + 后端 (Node.js + Prisma)
- Tab 结构：地图、照片库、手帐库、个人中心
- 地图 Tab：MapKit 足迹标注
- 照片库 Tab：照片导入、GPS 提取、CLGeocoder 反地理编码
- 手帐生成：DeepSeek API 驱动，8 套内置模板
- 手帐渲染：翻页模式 + 长图模式
- 个人中心：Apple 登录、DeepSeek API Key 设置
- 服务端：用户认证、旅行/手帐 CRUD、模板 API
